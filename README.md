# Jantrik Tools

Live Website: https://jantrik-tools.web.app/

## Features

- Slider banner on Homepage
- Various inventory items
- online order
- Check order
- Pay via Credit Card
- Add product
- Manage Order
- Payment implemented
- Login Authentication added (Email-Password & Google Sign In method implemented)
- Secured checkout area

## Technology used in Website

### Client Side

- React
- React bootstrap
- React firebase hooks
- React router dom
- React toastify
- React Stripe

### Server Side

- MongoDB
- Express.js
- Cors
- dotenv
